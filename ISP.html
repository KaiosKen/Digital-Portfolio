Merlin Run

My ISP, which I made using IGIS (Interactive Graphic Interface System), is a sidescrolling reaction game made by me, Muqadam Sabir, and Jonathan Gately. It was made using purely swift, and it was quite the challenge. We faced many issues, such as animation, collison, input, and physics. We eventually overcame these challenges and created an endless jumping game similar to the Google Dinosoaur Game.
